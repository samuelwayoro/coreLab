Ce package contient des bouts code qui explique ce que c'est que la programmation fonctionnelle . 
Il s'agit d'un nouveau paradigme introduit dans java avec sa version 8 et qui se différencie de 
l'approche traditionnelle dans laquelle le développeur se base sur la description des instructions
que l'ordinateur devra suivre pour obtenir le resultat. 
 
Alors que la programmation fonctionnnelle elle se base sur l'utilisation de fonctions qui nous servirons à
obtenir les resultats .

Pour réaliser une programmation fonctionnelle , le développeur doit utiliser au fur et à mésure dans son programme
des fonctions qui retourne chacune un résultat utilisé probablement par une/plusieurs autres fonctions afin d'en 
receuillir un résultat final. Ce paradigme est fortement basé sur le fonctions mathématique .
